使用gh-pages工具

首先构建静态页面：

gitbook build

下一个步骤使用了gh-pages这个工具，它可以将文件夹一键发布到 GitHub 项目下的gh-pages分支中。

首先先安装gh-pages工具：

npm install -g gh-pages

然后输入以下指令：

gh-pages -d _book

然后 _book 下的所有文档都会部署到gh-pages分支上。
